# ExpenseTracker API

## About Project

This project is a RESTful API for managing personal financial records. Users can record their income and expenses, retrieve past transactions,
and get summaries by category or time period.

## Technologies Used
- Node.js
- Express.js
- MongoDB
- Mongoose
- dotenv
- cors
- nodemon
